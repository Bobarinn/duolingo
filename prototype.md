Prototype Demo Description
Feature: Adaptive Chess Coach

The prototype demonstrates how an AI-powered coaching system can help Duolingo chess learners understand mistakes and improve faster.

Instead of only showing the correct move when a user makes a mistake, the system provides a short explanation and a follow-up puzzle designed to reinforce the concept the learner missed.

The goal of the feature is to make Duolingo feel more like a personal chess coach guiding the learner through patterns and tactics.

Although this demo focuses on chess, the same idea could also be applied to language lessons by explaining grammar mistakes and providing targeted practice.

Prototype Flow (4 Screens)

The prototype should demonstrate a simple step-by-step learning flow.

Screen 1 — Puzzle Attempt

The learner is solving a chess puzzle.

The screen shows:

A chessboard position

Text: “White to move”

The learner chooses a move

Example scenario:

The correct tactic is Nf7+ (a knight fork).

The learner instead plays Qh5, which is incorrect.

After the move, the system shows feedback:

❌ Not quite — let's break it down.

Button:

See Explanation

Screen 2 — AI Explanation

The system highlights the tactical idea the learner missed.

The chessboard remains visible with key squares highlighted.

Example explanation card:

“You moved your queen too early. The stronger move was Nf7+, which creates a knight fork attacking both the king and queen.”

Highlighted elements:

Knight square (Nf7)

King

Queen

This helps the learner visually understand the tactic.

Button:

Try a Similar Puzzle

Screen 3 — Follow-Up Puzzle

The learner receives a quick reinforcement puzzle with a similar tactic.

Example:

Text:

White to move — find the fork.

The board shows a similar tactical opportunity.

When the learner finds the correct move, the system shows encouragement:

✅ Nice! That's a knight fork.

Button:

Continue Learning

Screen 4 — Learning Insight

The system summarizes the learner’s pattern.

Example message:

“We noticed you sometimes miss fork opportunities. We'll include more fork puzzles in your next practice sessions.”

Progress indicator example:

Tactics Training
Forks — Improving

Button:

Return to Lesson

Goal of the Prototype

The prototype demonstrates how AI can:

Explain tactical mistakes clearly

Reinforce concepts with immediate practice

Personalize training based on learner mistakes

Instead of only showing the correct move, Duolingo becomes a guided learning experience similar to working with a chess coach.
